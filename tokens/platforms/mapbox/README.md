## How to use

### Upload to Mapbox Studio
1. https://studio.mapbox.com/styles/ ‚Üí **New style ‚Üí Upload**
2. Upload `style.light.json` (then separately `style.dark.json`)
3. Copy the two `mapbox://styles/<user>/<id>` URLs
4. Update `tokens/semantic/mapbox.tokens.json` with the new IDs
5. Run `pnpm tokens:build` to regenerate `tokens/platforms/{web,swift,kotlin}/mapbox.*`

### Use directly at runtime (Mapbox GL JS / native SDK)
Pass the file contents (or a URL serving them) as the `style` argument instead of a `mapbox://` URL. Web example:

```ts
import light from '@/tokens/platforms/mapbox/style.light.json'
import dark  from '@/tokens/platforms/mapbox/style.dark.json'

new mapboxgl.Map({ container, style: isDark ? dark : light })
```

iOS (`MapboxMaps`) and Android (`MapboxMaps`) both accept a JSON style string via `Style.loadStyle(.json(...))` / `MapboxStyleManager.loadStyle(styleJson)`.

## Layer structure

Both styles share the same layer order (bottom ‚Üí top):

1. `background` ‚Äî land/paper
2. Landuse fills ‚Äî national park, park/pitch/grass, hospital, school
3. Hillshade + contour lines (topographic feel)
4. `water`, `waterway`
5. `aeroway-runway`
6. `building` (zoom ‚â• 14, fades in 14 ‚Üí 15)
7. Road network, low ‚Üí high class:
   - `road-path` (dashed)
   - `road-street` + `road-street-casing`
   - `road-secondary-tertiary`
   - `road-primary`
   - `road-trunk-casing` + `road-trunk` (copper)
   - `bridge-trunk`
8. Admin boundaries ‚Äî state/province (dashed), country
9. Labels ‚Äî road, POI, neighborhood (UPPERCASE, tracked), locality, city, state, country

The motorway/trunk copper accent is the single branded flourish ‚Äî everything else reads as paper/ink topography so that overlaid route polylines, POIs, and annotations remain the loudest element on the map.

## Fonts

Uses Mapbox's built-in `DIN Offc Pro` stack (Regular / Medium / Bold) with `Arial Unicode MS` fallback. If you swap to a LaneShadow-custom font stack in Studio, update the `text-font` arrays in both files.

## Regenerating

These files are **hand-authored against the theme tokens** ‚Äî they are not auto-generated by `tokens/scripts/generate.ts`. When tokens drift, update the palette table above and the embedded hex values in both files, then re-upload to Studio.
