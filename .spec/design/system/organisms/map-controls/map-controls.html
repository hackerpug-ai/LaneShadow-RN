org-map-controls

Right-side vertical workbar · zoom · recenter · layers · save · mode toggle · Mirrors react-native MapControls

State Variants — Light

--map mode · no route

Default state for idle / planning / sessions / error. Zoom + recenter + layers + chat-mode toggle. Save chip hidden (no route).

--map mode · with route

Route-results / route-details state. Save bookmark chip appears below layers.

--map mode · saved (favorited)

Save chip flips to copper signal fill when isSavedRoute is true. Bookmark glyph filled.

--chat mode

When the rider opens the chat-mode view, all map-specific controls collapse and the toggle slot becomes a "back to map" chip.

State Variants — Dark

--map mode · no route · dark

Glass surface re-resolves to dark. Stroke-md icons remain content-primary on dark.

--map mode · saved · dark

Copper signal save state remains identical brand color across themes.
