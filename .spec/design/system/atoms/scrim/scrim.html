Touch behavior — non-blocking vs blocking
Touch-through (default)
pointer-events: none
Taps pass through to map controls beneath. Default for LSSessionsDrawer — rider can still interact with the visible map strip.
Blocking (explicit opt-in)
pointer-events: auto
Captures taps, fires onTap callback. Required for BottomSheet (0.50) and Modal (0.60 / 0.76). Must be explicitly enabled — never default.

opacity.scrim.opaque 0.95 0.95 Full interrupt. Breaks spatial continuity — use sparingly.
