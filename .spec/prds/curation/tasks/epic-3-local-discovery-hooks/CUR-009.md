## BACKGROUND

With the `discovery.db` schema and lean-tier sync in place (CUR-008), this task builds the SQL query layer that drives all local discovery. The TRD (Section 6.2) defines four query patterns: `queryNearby` (bounding box), `queryByState`, `queryByArchetype`, and `queryTopRoutes`. All queries use parameterized SQL, ORDER BY the pre-computed `composite_score`, and leverage the centroid indexes for performance.

These queries are the sole data access layer for discovery — no LLM is involved in query construction or ranking (P1 constraint). Sorting is always by `composite_score` (precomputed at ingestion), never by values computed at query time.

**PRD references:** TRD-6.2 (Query Patterns), P1 (LLMs do text → structure, never structure → selection), P5 (deterministic parser between LLM and downstream).

import type { DB, CuratedRouteLean } from './db';

export interface QueryOptions {
  archetype?: string;
  limit?: number;

export function queryNearby(
  db: DB,
  lat: number,
  lng: number,
  radiusDeg: number = 0.5,
  opts: QueryOptions = {}
): CuratedRouteLean[] {
  const { archetype, limit = 50 } = opts;
  const sql = archetype
    ? `SELECT * FROM curated_routes
       WHERE centroid_lat BETWEEN ? AND ?
       AND centroid_lng BETWEEN ? AND ?
       AND primary_archetype = ?
       ORDER BY composite_score DESC
       LIMIT ?`
    : `SELECT * FROM curated_routes
       WHERE centroid_lat BETWEEN ? AND ?
       AND centroid_lng BETWEEN ? AND ?
       ORDER BY composite_score DESC
       LIMIT ?`;

    ? [lat - radiusDeg, lat + radiusDeg, lng - radiusDeg, lng + radiusDeg, archetype, limit]

## NOTES

- The `radiusDeg` parameter in `queryNearby` defaults to ~0.5 degrees (~35-50 miles depending on latitude), matching the 50-mile default radius in UC-DISC-01.
- Distance-based sorting ("Nearest") is handled at the hook level (CUR-010) by computing haversine distance from the user's location to each route centroid — not in SQL.
- All archetype values are validated against the enum: `twisties | mountain | coastal | adventure | scenic_byway | desert`.
