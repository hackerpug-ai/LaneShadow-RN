## ACCEPTANCE CRITERIA

### AC-001: Happy Path — Offline Route Creation and Sync
**GIVEN** user is offline with downloaded map region
**WHEN** user creates route "SF to Santa Cruz" with waypoints
**THEN** route saves locally AND leg labels generate immediately (0.35s) AND when user goes online, route syncs to server automatically

### AC-002: Edge Case — Concurrent Edit Sync
**GIVEN** user edits route name on phone (offline)
**WHEN** user simultaneously edits same route on tablet (offline)
**THEN** both devices go online AND both changes sync AND Yjs CRDT merges correctly (last-write-wins for same field)

### AC-003: Edge Case — Progressive Enhancement Flow
**GIVEN** user creates new route
**WHEN** enrichment starts
**THEN** leg labels appear immediately (local Qwen3.5) AND creative labels fade in after 3.9s (Haiku) AND weather overlays appear after 20s AND UI updates smoothly at each stage

### AC-004: Error Case — Sync Failure Recovery
**GIVEN** user has pending route edits offline
**WHEN** user goes online but server returns 500 error
**THEN** edits remain in local storage AND sync retries automatically with exponential backoff AND sync succeeds on second attempt

### Offline Routing Test
```typescript
// e2e/tests/offline-routing.test.ts
import { test, expect } from "@playwright/test";

test.describe("offline routing", () => {
  test("creates route offline and syncs when online", async ({ page }) => {
    // Enable airplane mode
    await page.goto("/settings");
    await page.click('[data-testid="airplane-mode-toggle"]');
    await page.click('[data-testid="wifi-toggle"]', { state: "hidden" });

    // Create route
    await page.goto("/routes/new");
    await page.fill('[name="start"]', "San Francisco");
    await page.fill('[name="end"]', "Santa Cruz");
    await page.click('[data-testid="calculate-route"]');

    // Verify leg labels appear immediately
    await expect(page.locator('[data-testid="leg-label"]')).toBeVisible();
    await expect(page.locator('[data-testid="leg-label"]')).toHaveText(/FROM → TO/);

    // Disable airplane mode
    await page.goto("/settings");
    await page.click('[data-testid="airplane-mode-toggle"]');
    await page.click('[data-testid="wifi-toggle"]');

    // Verify sync completes
    await expect(page.locator('[data-testid="sync-status"]')).toHaveText("Synced");
  });
});
```

### Sync Flow Test
```typescript
// e2e/tests/sync-flow.test.ts
test.describe("sync flow", () => {
  test("merges concurrent edits correctly", async ({ page, context }) => {
    // Create route on device 1
    await page.goto("/routes/test-route");
    await page.fill('[name="route-name"]', "Coastal Run");
    await page.click('[data-testid="save-route"]');

    // Simulate concurrent edit on device 2
    const page2 = await context.newPage();
    await page2.goto("/routes/test-route");
    await page2.fill('[name="route-description"]', "Scenic coastal drive");

    // Both save
    await Promise.all([
      page.click('[data-testid="save-route"]'),
      page2.click('[data-testid="save-route"]'),
    ]);

    // Verify merge
    await page.reload();
    await expect(page.locator('[name="route-name"]')).toHaveValue("Coastal Run");
    await expect(page.locator('[name="route-description"]')).toHaveValue("Scenic coastal drive");
  });
});
```
