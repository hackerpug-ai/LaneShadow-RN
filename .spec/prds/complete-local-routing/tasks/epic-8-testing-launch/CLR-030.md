## DESIGN

**References:**
- Markdown Guide: https://www.markdownguide.org/
- ADR Template: https://adr.github.io/
- PRD Documentation: `.spec/prds/complete-local-routing/`
- Technical Writing Best Practices: Clear, concise, example-driven

**Patterns:**
- Markdown formatting with code blocks
- Step-by-step instructions
- Code examples with syntax highlighting
- Troubleshooting flowcharts
- Architecture decision records (ADRs)

**Anti-Patterns:**
- **DO NOT** write docs without code examples
- **DO NOT** assume reader has context
- **DO NOT** skip troubleshooting steps
- **DO NOT** document outdated information
- **DO NOT** use jargon without explanation

**Migration Guide Structure:**
```markdown
# Migration Guide: Google Maps to Mapbox

## Coordinate System Conversion
### Google: [lat, lng]
### Mapbox: [lng, lat]

### Code Example
\`\`\`typescript
// Before (Google)
const coord = [37.7749, -122.4194]  // [lat, lng]

// After (Mapbox)
const coord = [-122.4194, 37.7749]  // [lng, lat]
\`\`\`

## Breaking Changes
- ...
```

**ADR Template:**
```markdown
# ADR 001: Mapbox SDK Selection

## Status
Accepted

## Context
We need to migrate from Google Maps to reduce costs.

## Decision
Use @rnmapbox/maps SDK for offline routing.

## Rationale
- Proven technology (BMW, Weather Channel)
- True offline routing without native modules
- React Native compatible (no custom bridge)

## Consequences
- Positive: $1,470/month savings
- Negative: Coordinate system conversion required
```

## NOTES

**Documentation Principles:**
- **Clear:** Use plain language, avoid jargon
- **Concise:** Get to the point quickly
- **Example-driven:** Show, don't just tell
- **Maintainable:** Keep docs in sync with code

**Migration Guide Highlights:**
- Coordinate system: [lat, lng] → [lng, lat]
- Polyline encoding: Google polyline → Mapbox polyline6
- API changes: MapView props, camera controls, event handlers
- Testing: Verify on physical devices (not just simulators)

**User Guide Highlights:**
- WiFi requirement: Cellular downloads blocked
- Region sizes: City (50MB), Mountain (800MB)
- Download process: Settings → Offline Maps → Download
- Troubleshooting: Check WiFi, storage, restart app

- Google Maps: $1,500/month at 1,000 routes/day
- Mapbox: $15/month (50k directions)
- Haiku: $15/month (12.5% of routes, down from 100%)

- **Total savings: ~$1,470/month**
