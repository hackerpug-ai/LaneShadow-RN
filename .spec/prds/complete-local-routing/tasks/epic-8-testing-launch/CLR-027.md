## CRITICAL CONSTRAINTS

1. **MUST** Test end-to-end user flows on physical devices — NO simulator/emulator tests
2. **NEVER** mock Mapbox SDK or Replicate sync — test actual integrations
3. **STRICTLY** Test offline→online→offline transitions — CRITICAL for local-first architecture
4. **MUST** Test concurrent multi-device sync — verify Yjs CRDT merge behavior
5. **NEVER** ship without testing model download failure and recovery — highest user risk

## SPECIFICATION

Execute comprehensive integration testing across all epics, validating end-to-end user flows including offline routing, sync, progressive enrichment, and weather overlays on real iOS and Android devices. Success state: All critical user flows pass integration tests, edge cases are covered, and app is production-ready.

---

## ACCEPTANCE CRITERIA

### AC-001: Happy Path — Complete User Flow
**GIVEN** new user launches app for first time
**WHEN** user completes setup, downloads region, creates route, and syncs
**THEN** all steps complete successfully AND no crashes occur AND data persists correctly

### AC-002: Edge Case — Model Download Failure and Recovery
**GIVEN** user is downloading model and connection fails at 60%
**WHEN** user reconnects to WiFi
**THEN** download resumes from 60% AND completes successfully AND model loads correctly

### AC-003: Edge Case — Multi-Device Concurrent Edit
**GIVEN** user has route open on phone and tablet
**WHEN** user edits route name on phone and waypoints on tablet simultaneously
**THEN** both edits sync AND Yjs CRDT merges correctly AND no data is lost

### AC-004: Error Case — Storage Exhaustion During Download
**GIVEN** user has 1GB free storage and attempts to download 2GB map region
**WHEN** download reaches storage limit
**THEN** download pauses gracefully AND user sees clear error message AND app doesn't crash

### Test Execution Pattern
```typescript
// e2e/integration/epic-01-shadow-setup.test.ts
import { test, expect } from "@playwright/test";

test.describe("Epic 1: Shadow Setup", () => {
  test("complete setup flow", async ({ page }) => {
    // 1. First launch
    await page.goto("/");
    await expect(page.locator('[data-testid="welcome-screen"]')).toBeVisible();

    // 2. WiFi check
    await expect(page.locator('[data-testid="wifi-required"]')).toBeVisible();

    // 3. Download model
    await page.click('[data-testid="download-shadow"]');
    await expect(page.locator('[data-testid="download-progress"]')).toBeVisible();

    // 4. Wait for completion
    await expect(page.locator('[data-testid="setup-complete"]')).toBeVisible({ timeout: 300000 });

    // 5. Access app
    await expect(page.locator('[data-testid="main-map"]')).toBeVisible();
  });
});
```
