## CRITICAL CONSTRAINTS

1. **MUST** use secure token storage - never hardcode Mapbox access tokens in source code
2. **STRICTLY** separate dev/prod environments - prevent production data in development
3. **MUST** configure production certificates correctly - app store submission requirement
4. **NEVER** commit .env files or secrets to repository - security requirement
5. **MUST** TestFlight build passes all validation checks - App Store requirement

---

## SPECIFICATION

**Objective:** Configure production build settings including Mapbox token provisioning, environment variables, production certificates, and App Store Connect configuration.

**Success State:** Production builds successfully for iOS and Android, Mapbox token is securely configured, environment variables are properly separated, certificates are provisioned, and TestFlight is ready for beta testing.

## DESIGN

**References:**
- Expo Environment Variables: https://docs.expo.dev/guides/environment-variables/
- iOS Distribution Guide: https://developer.apple.com/documentation/xcode/distributing-your-app-for-beta-testing-and-releases
- Android App Signing: https://developer.android.com/studio/publish/app-signing
- Mapbox Access Tokens: https://account.mapbox.com/

**Patterns:**
- Environment variable loading via process.env
- Separate .env files for dev/prod
- Secure certificate storage (never committed)
- App Store Connect configuration
- TestFlight beta testing

**Anti-Patterns:**
- **DO NOT** hardcode tokens in source code
- **DO NOT** commit .env files to repository
- **DO NOT** use development tokens in production
- **DO NOT** skip certificate validation
- **DO NOT** ignore build validation errors

**Environment Variable Configuration:**
```javascript
// app.config.js
export default {
  expo: {
    extra: {
      mapboxAccessToken: process.env.MAPBOX_ACCESS_TOKEN,
      convexUrl: process.env.CONVEX_URL,
      environment: process.env.APP_ENV || 'development'
    }
  }
}
```

**.env Files (gitignored):**
```bash
# .env.development
MAPBOX_ACCESS_TOKEN=pk.eyJ1Ijoi...
CONVEX_URL=https://lane-shadow-dev.convex.cloud
APP_ENV=development

# .env.production
MAPBOX_ACCESS_TOKEN=pk.eyJ1Ijoi...
CONVEX_URL=https://lane-shadow.convex.cloud
APP_ENV=production
```
