## ACCEPTANCE CRITERIA

### AC-001: Happy Path — Route Calculation Performance
**GIVEN** device has offline map region downloaded
**WHEN** user calculates route from SF to Santa Cruz
**THEN** route calculation completes in <2s AND geometry renders smoothly at 60fps

### AC-002: Edge Case — Local Model Memory Profiling
**GIVEN** Qwen3.5 0.8B model is loaded in memory
**WHEN** inference runs for 10 consecutive leg label generations
**THEN** peak memory usage remains <1.5GB AND memory doesn't leak between inferences

### AC-003: Edge Case — Progressive Loading Performance
**GIVEN** user creates new route on slow connection (3G)
**WHEN** enrichment starts
**THEN** leg labels appear in <0.5s (local) AND creative labels fade in by 5s AND route is usable within 10s total

### AC-004: Error Case — Performance Regression Detection
**GIVEN** baseline metrics exist from CLR-025
**WHEN** new code changes are committed
**THEN** CI benchmark runs AND regressions >10% fail build AND alert is posted to PR

### Route Calculation Benchmark
```typescript
// performance/benchmarks/routing.ts
import { bench, describe } from "vitest";

describe("route calculation", () => {
  bench("offline route calculation", async () => {
    const result = await calculateRoute({
      start: { lat: 37.7749, lng: -122.4194 }, // San Francisco
      end: { lat: 36.9741, lng: -122.0262 },   // Santa Cruz
      offline: true,
    });
    expect(result.geometry).toBeDefined();
  }, { time: 2000 }); // Must complete in 2s

  bench("online route calculation", async () => {
    const result = await calculateRoute({
      start: { lat: 37.7749, lng: -122.4194 },
      end: { lat: 36.9741, lng: -122.0262 },
      offline: false,
    });
    expect(result.geometry).toBeDefined();
  }, { time: 1000 }); // Must complete in 1s
});
```

### Model Inference Benchmark
```typescript
// performance/benchmarks/model-inference.ts
describe("local model inference", () => {
  bench("leg label generation", async () => {
    const result = await generateLegLabel({
      from: "San Francisco",
      to: "Santa Cruz",
    });
    expect(result.label).toMatch(/FROM → TO/);
  }, { time: 500 }); // Must complete in 0.5s

  bench("memory usage - 10 inferences", async () => {
    const startMemory = process.memoryUsage().heapUsed;
    for (let i = 0; i < 10; i++) {
      await generateLegLabel({ from: "City A", to: "City B" });
    }
    const endMemory = process.memoryUsage().heapUsed;
    const memoryMB = (endMemory - startMemory) / 1024 / 1024;
    expect(memoryMB).toBeLessThan(1500); // <1.5GB
  });
});
```
