# CLR-012: Replicate Integration Setup

**Task ID:** CLR-012
**Epic:** Epic 5 - Route Sync
**Agent:** convex-implementer
**Estimated Time:** 480 min (8 hours)
**Status:** Pending

---

## CRITICAL CONSTRAINTS

1. **MUST** Use @trestleinc/replicate ^1.x for local-first sync — no other CRDT libraries
2. **NEVER** run Replicate without crypto polyfills — react-native-get-random-values is MANDATORY
3. **STRICTLY** install @op-engineering/op-sqlite ^7.x — Replicate's ONLY supported SQLite engine for React Native
4. **MUST** configure Replicate component in convex/convex.config.ts BEFORE any collection usage
5. **NEVER** mix Replicate collections with standard Convex collections — all synced data MUST use Replicate

---

## SPECIFICATION

Install and configure @trestleinc/replicate for local-first route synchronization with automatic conflict resolution via Yjs CRDTs, enabling offline route editing with bidirectional sync when online. Success state: Replicate is installed, crypto polyfills configured, op-sqlite integrated, and Replicate component registered in Convex config.

---

## ACCEPTANCE CRITERIA

### AC-001: Happy Path — Successful Installation and Configuration
**GIVEN** fresh React Native + Expo project with Convex backend
**WHEN** developer installs @trestleinc/replicate, @op-engineering/op-sqlite, and crypto polyfills
**THEN** all packages install without version conflicts AND Replicate component appears in convex.config.ts AND app builds successfully for iOS and Android

### AC-002: Edge Case — Crypto Polyfill Missing
**GIVEN** @trestleinc/replicate is installed but react-native-get-random-values is NOT installed
**WHEN** app attempts to use Replicate collection
**THEN** clear error message displays: "Missing crypto polyfill. Install react-native-get-random-values and import in app entry point." AND app doesn't crash

### AC-003: Edge Case — Wrong SQLite Engine
**GIVEN** developer attempts to use @react-native-async-storage/async-storage instead of @op-engineering/op-sqlite
**WHEN** Replicate initializes storage
**THEN** initialization fails with error: "op-sqlite is required. Installed: async-storage" AND documentation link is provided

### AC-004: Error Case — Replicate Component Not Registered
**GIVEN** developer creates use-routes.ts collection but forgets to add Replicate component to convex.config.ts
**WHEN** app attempts to sync routes
**THEN** runtime error occurs: "Replicate component not found in convex.config.ts. Add: components: [{ replicate: {} }]" AND app doesn't silently fail

## DESIGN

### Installation Commands
```bash

npm install @op-engineering/op-sqlite

# Crypto polyfills (REQUIRED)
```
