## Task Description

Design weather color mappings for dark/light themes that render weather data (wind, rain, temperature) on route polylines. Colors must map to severity/intensity levels, maintain semantic meaning, pass WCAG AA contrast requirements, and work consistently across both themes.

---

## Human Test Deliverable

**Test Scenario:** Rider views route with weather conditions

**Steps:**
1. Open route with mixed weather conditions
2. See wind levels on route (green → yellow → red)
3. See rain segments (light blue → dark blue)
4. See temperature colors (blue → copper → red)
5. Switch from dark to light theme
6. Verify all weather colors still readable
7. Verify semantic meaning preserved (green = safe, red = dangerous)
8. Zoom in/out and verify colors visible at all scales

**Gate:** Weather render < 100ms, Contrast ≥ 4.5:1 (WCAG AA)

## Specification

### Color Mappings to Define

#### 1. Wind Levels
**Severity:** Low → Medium → High

**Dark Theme:**
- Low (calm): `#31A362` (green, 65% saturation)
- Medium (moderate): `#F59E0B` (amber/warning)
- High (dangerous): `#E35D6A` (red/coral)

**Light Theme:**
- Low (calm): `#268A4D` (darker green for contrast)
- Medium (moderate): `#D98E04` (darker amber)
- High (dangerous): `#C94352` (darker red)

**Semantic Mapping:**
```typescript
wind: {
  low: semantic.color.success.default,      // Green - safe
  medium: semantic.color.warning.default,    // Amber - caution
  high: semantic.color.danger.default,       // Red - dangerous
}
```

#### 2. Rain Intensity
**Severity:** None → Light → Moderate → Heavy

**Dark Theme:**
- None: Transparent (no overlay)
- Light: `#4FC3F7` (light blue, 75% opacity)
- Moderate: `#29B6F6` (medium blue, 85% opacity)
- Heavy: `#0288D1` (dark blue, 95% opacity)

**Light Theme:**
- None: Transparent
- Light: `#81D4FA` (lighter blue, 75% opacity)
- Moderate: `#29B6F6` (medium blue, 85% opacity)
- Heavy: `#0277BD` (darker blue, 95% opacity)

**Semantic Mapping:**
```typescript
rain: {
  light: 'rgba(79, 195, 247, 0.75)',   // Light blue
  moderate: 'rgba(41, 182, 246, 0.85)', // Medium blue
  heavy: 'rgba(2, 136, 209, 0.95)',     // Dark blue
}
```

**Opacity Strategy:**
- Rain colors use opacity to allow route geometry to show through
- Higher intensity = higher opacity (more prominent)
- 75-95% range balances visibility with route context

#### 3. Temperature
**Comfort:** Cold → Mild → Hot

**Dark Theme:**
- Cold: `#2B9AEB` (blue, information)
- Mild: `#B87333` (copper, brand color)
- Hot: `#E35D6A` (red, danger)

**Light Theme:**
- Cold: `#1081D6` (darker blue for contrast)
- Mild: `#B87333` (same copper)
- Hot: `#C94352` (darker red)

**Semantic Mapping:**
```typescript
temperature: {
  cold: semantic.color.info.default,       // Blue - cold
  mild: semantic.color.primary.default,    // Copper - comfortable
  hot: semantic.color.danger.default,      // Red - hot
}
```

**Rationale:**
- Copper maps to "mild" as brand color
- Blue (info) maps to cold
- Red (danger) maps to hot
- Creates semantic temperature gradient

### Theme Adjustments

#### Dark Theme Strategy
- **Higher saturation:** Colors more vibrant against dark background
- **Slightly lighter values:** Better visibility on dark surfaces
- **Lower opacity for overlays:** 75-95% range for rain

#### Light Theme Strategy
- **Lower saturation:** Colors less intense on bright background
- **Slightly darker values:** Better contrast against light surfaces
- **Same opacity range:** Consistent with dark theme

## Design Requirements

### Theme Compliance
- **Dark-first design:** Design colors for dark mode first
- **Semantic meaning:** Green = safe, Red = dangerous (universal)
- **Copper accent:** Use brand color for mild/comfortable states
- **Consistent with existing UI:** Match weather badge colors

See: `styles/RULES.md:line:17`

### WCAG AA Compliance
- **Contrast ratio:** ≥ 4.5:1 for normal text
- **Route overlay:** ≥ 3:1 against map background
- **Color blindness:** Test with deuteranopia/protanopia filters

### Semantic Consistency
- **Wind:** Match existing wind badge colors
- **Rain:** Match existing rain badge colors
- **Temperature:** Match existing temperature badge colors

### Visual Hierarchy
- **High severity:** Most prominent (red, opaque)
- **Medium severity:** Moderate prominence (amber, 85% opacity)
- **Low severity:** Subtle (green, 75% opacity)

## Examples

### Weather Color Constants

```typescript
// styles/weather.ts
import { SemanticTheme } from './types'

export const getWeatherColors = (semantic: SemanticTheme) => ({
  wind: {
    low: semantic.color.success.default,
    medium: semantic.color.warning.default,
    high: semantic.color.danger.default,
  },

  rain: {
    light: 'rgba(79, 195, 247, 0.75)',
    moderate: 'rgba(41, 182, 246, 0.85)',
    heavy: 'rgba(2, 136, 209, 0.95)',
  },

  temperature: {
    cold: semantic.color.info.default,
    mild: semantic.color.primary.default,  // Copper
    hot: semantic.color.danger.default,
  },

  // Mapbox line layer style factory
  toLineLayerStyle: (color: string, opacity: number) => ({
    lineColor: color,
    lineWidth: 5,
    lineOpacity: opacity,
    lineCap: 'round',
    lineJoin: 'round',
  }),
})
```

### Weather Overlay Usage

```typescript
// components/map/weather-overlay.tsx
import { getWeatherColors } from '@/styles/weather'
import { useSemanticTheme } from '@/hooks/use-semantic-theme'

export const WeatherOverlay: React.FC<{ route: Route }> = ({ route }) => {
  const { semantic } = useSemanticTheme()
  const colors = getWeatherColors(semantic)

  return (
    <MapboxMapView.ShapeSource
      id="weather-source"
      shape={formatWeatherGeoJSON(route.weather)}
    >
      {/* Wind layer */}
      <MapboxMapView.LineLayer
        id="wind-layer"
        style={{
          lineColor: ['match',
            ['get', 'windLevel'],
            'low', colors.wind.low,
            'medium', colors.wind.medium,
            'high', colors.wind.high,
            colors.wind.medium, // default
          ],
          lineWidth: 5,
          lineOpacity: 1.0,
        }}
      />

      {/* Rain layer */}
      <MapboxMapView.LineLayer
        id="rain-layer"
        style={{
          lineColor: ['match',
            ['get', 'rainIntensity'],
            'light', colors.rain.light,
            'moderate', colors.rain.moderate,
            'heavy', colors.rain.heavy,
            colors.rain.light, // default
          ],
          lineWidth: 5,
          lineOpacity: ['match',
            ['get', 'rainIntensity'],
            'light', 0.75,
            'moderate', 0.85,
            'heavy', 0.95,
            0.75, // default
          ],
        }}
      />

      {/* Temperature layer */}
      <MapboxMapView.LineLayer
        id="temp-layer"
        style={{
          lineColor: ['match',
            ['get', 'temperature'],
            'cold', colors.temperature.cold,
            'mild', colors.temperature.mild,
            'hot', colors.temperature.hot,
            colors.temperature.mild, // default
          ],
          lineWidth: 5,
          lineOpacity: 1.0,
        }}
      />
    </MapboxMapView.ShapeSource>
  )
}
```

### Color Mapping Validation

```typescript
// __tests__/weather-colors.test.ts
import { getWeatherColors } from '@/styles/weather'
import { lightTheme, darkTheme } from '@/styles/theme'

describe('Weather Color Mappings', () => {
  describe('Dark Theme', () => {
    const colors = getWeatherColors(darkTheme.semantic)

    test('wind colors map to severity', () => {
      expect(colors.wind.low).toBe('#31A362') // Green
      expect(colors.wind.medium).toBe('#F59E0B') // Amber
      expect(colors.wind.high).toBe('#E35D6A') // Red
    })

    test('rain colors have correct opacity', () => {
      expect(colors.rain.light).toContain('0.75')
      expect(colors.rain.moderate).toContain('0.85')
      expect(colors.rain.heavy).toContain('0.95')
    })

    test('temperature colors map to comfort', () => {
      expect(colors.temperature.cold).toBe('#2B9AEB') // Blue
      expect(colors.temperature.mild).toBe('#B87333') // Copper
      expect(colors.temperature.hot).toBe('#E35D6A') // Red
    })
  })

  describe('Light Theme', () => {
    const colors = getWeatherColors(lightTheme.semantic)

    test('colors have sufficient contrast', () => {
      // WCAG AA requires ≥ 4.5:1 for normal text
      // Route overlays require ≥ 3:1 against map
      expect(getContrastRatio(colors.wind.high, '#F5F0EB')).toBeGreaterThanOrEqual(3.0)
      expect(getContrastRatio(colors.temperature.hot, '#FFFFFF')).toBeGreaterThanOrEqual(3.0)
    })
  })

  describe('Semantic Consistency', () => {
    test('colors match existing badge colors', () => {
      const semantic = darkTheme.semantic
      const colors = getWeatherColors(semantic)

      expect(colors.wind.low).toBe(semantic.color.success.default)
      expect(colors.wind.medium).toBe(semantic.color.warning.default)
      expect(colors.wind.high).toBe(semantic.color.danger.default)
    })
  })
})
```
