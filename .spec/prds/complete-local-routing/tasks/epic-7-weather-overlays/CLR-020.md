## DESIGN

**References:**
- Mapbox ShapeSource Documentation: https://github.com/rnmapbox/maps/docs/shapesource.md
- Mapbox LineLayer Documentation: https://github.com/rnmapbox/maps/docs/linelayer.md
- PRD Phase 4: Weather Overlay Implementation - "Weather ShapeSource Rendering" section
- CLR-021 (Theme Color Mapping) - Provides color mappings

**Patterns:**
- ShapeSource + LineLayer pattern for polyline rendering
- GeoJSON FeatureCollection format
- Layer ordering with before/after properties
- Zoom-based styling with expressions
- Conditional rendering for missing data

**Anti-Patterns:**
- **DO NOT** render weather below route geometry
- **DO NOT** block UI thread during rendering
- **DO NOT** ignore zoom level (details won't be visible)
- **DO NOT** crash on missing weather data
- **DO NOT** use complex JS logic (use native performance)

**WeatherOverlay Component:**
```typescript
interface WeatherOverlayProps {
  route: Route
  weather: WeatherData
  visibleLayers: {
    wind: boolean
    rain: boolean
    temp: boolean
  }
}

export const WeatherOverlay: React.FC<WeatherOverlayProps> = ({
  route,
  weather,
  visibleLayers
}) => (
  <>
    <MapboxMapView.ShapeSource
      id="weather-source"
      shape={formatWeatherGeoJSON(route, weather)}
    >
      {visibleLayers.wind && (
        <MapboxMapView.LineLayer
          id="wind-layer"
          style={windLayerStyle}
          belowLayerId="rain-layer"
        />
      )}
      {visibleLayers.rain && (
        <MapboxMapView.LineLayer
          id="rain-layer"
          style={rainLayerStyle}
          belowLayerId="temp-layer"
        />
      )}
      {visibleLayers.temp && (
        <MapboxMapView.LineLayer
          id="temp-layer"
          style={tempLayerStyle}
        />
      )}
    </MapboxMapView.ShapeSource>
  </>
)
```

**Layer Style Configuration:**
```typescript
const windLayerStyle = {
  lineColor: ['get', 'windColor'],
  lineWidth: 4,
  lineOpacity: 0.8
}

const rainLayerStyle = {
  lineColor: ['get', 'rainColor'],
  lineWidth: 6,
  lineOpacity: 0.6
}

const tempLayerStyle = {
  lineColor: ['get', 'tempColor'],
  lineWidth: 3,
  lineOpacity: 0.5
}
```

## NOTES

**Layer Ordering:**
1. Route geometry (bottom)
2. Wind layer
3. Rain layer
4. Temperature layer (top)

**Zoom-Based Styling:**
- Low zoom (country level): reduced detail, thicker lines
- High zoom (street level): full detail, thinner lines
- Use Mapbox expressions for zoom-based properties

**Color Mappings:**
- Wind: green (low) → yellow (medium) → red (high)
- Rain: light blue (light) → dark blue (heavy)
- Temperature: blue (cold) → copper (moderate) → red (hot)

**Performance Optimization:**
- Use native ShapeSource (not JS rendering)
- Batch LineLayer updates when possible
- Simplify geometry for long routes
- Test on physical devices (simulator performance differs)
