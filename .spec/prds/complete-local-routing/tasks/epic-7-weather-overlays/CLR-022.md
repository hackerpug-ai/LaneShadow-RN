## DESIGN

**References:**
- Douglas-Peucker Algorithm: https://en.wikipedia.org/wiki/Ramer–Douglas–Peucker_algorithm
- Mapbox Performance Tips: https://docs.mapbox.com/android/maps/guides/performance/
- PRD Phase 4: Weather Overlay Implementation - "Batch Rendering Optimization" section

**Patterns:**
- Geometry batching by weather type
- Level-of-detail (LOD) based on zoom
- Douglas-Peucker simplification algorithm
- Memory profiling and monitoring
- Performance benchmarking

**Anti-Patterns:**
- **DO NOT** over-simplify geometry (route becomes unrecognizable)
- **DO NOT** skip LOD transitions (will look jerky)
- **DO NOT** ignore memory leaks (will crash app)
- **DO NOT** optimize solely on simulator (device performance differs)
- **DO NOT** sacrifice visual accuracy for speed

**Batching Algorithm:**
```typescript
export const batchWeatherPolylines = (
  segments: WeatherSegment[],
  zoom: number
): GeoJSON.FeatureCollection => {
  // Group by weather type
  const byType = groupBy(segments, s => s.weather.type)

  // Simplify based on LOD
  const lod = calculateLOD(zoom)
  const simplified = Object.entries(byType).map(([type, segments]) =>
    simplifySegments(segments, lod)
  )

  return {
    type: 'FeatureCollection',
    features: simplified.flatMap(createFeature)
  }
}

const calculateLOD = (zoom: number): number => {
  // Higher zoom = more detail
  // zoom 10 (country): tolerance 0.001
  // zoom 15 (city): tolerance 0.0001
  // zoom 18 (street): tolerance 0 (no simplification)
  return Math.max(0, 0.001 * Math.pow(10, 15 - zoom))
}
```

**Geometry Simplification:**
```typescript
export const simplifyDouglasPeucker = (
  points: [number, number][],
  tolerance: number
): [number, number][] => {
  if (points.length <= 2) return points

  // Find point with maximum distance from line
  let maxDist = 0
  let index = 0
  const end = points.length - 1

  for (let i = 1; i < end; i++) {
    const dist = perpendicularDistance(points[i], points[0], points[end])
    if (dist > maxDist) {
      maxDist = dist
      index = i
    }
  }

  // If max distance is greater than tolerance, recursively simplify
  if (maxDist > tolerance) {
    const left = simplifyDouglasPeucker(points.slice(0, index + 1), tolerance)
    const right = simplifyDouglasPeucker(points.slice(index), tolerance)
    return [...left.slice(0, -1), ...right]
  } else {
    return [points[0], points[end]]
  }
}
```
