# US-028: Build WeatherTimelineSheet for hourly detail

> Task ID: US-028
> Type: FEATURE
> Priority: P1
> Estimate: 120 minutes
> Assignee: ui-developer

## CRITICAL CONSTRAINTS

### MUST
- Use BottomSheetWrapper as container
- Hourly rows showing: time, condition icon, temperature, wind speed, rain percentage
- Align rows to departure time
- Highlight worst hour with warning icon
- Peak Conditions summary at bottom
- Collapsible sheet behavior
- onAdjustDeparture callback for departure time changes

### NEVER
- Use a modal or full-screen view (must be a bottom sheet)
- Show hours outside the ride window
- Skip the worst-hour highlight

### STRICTLY
- Rows ordered chronologically from departure time
- Worst hour = hour with lowest conditions score

## SPECIFICATION

**Objective:** Build an expandable bottom sheet that shows hourly weather detail for a selected route, aligned to the planned departure time.

**Success looks like:** Tapping expand on a route card opens a bottom sheet with hourly weather rows. The worst hour is highlighted. A Peak Conditions summary shows at the bottom. The sheet is collapsible and provides a callback for adjusting departure time.
