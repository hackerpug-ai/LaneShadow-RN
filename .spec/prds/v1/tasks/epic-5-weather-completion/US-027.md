# US-027: Add weather badges to RouteAttachmentCard

> Task ID: US-027
> Type: FEATURE
> Priority: P0
> Estimate: 90 minutes
> Assignee: ui-developer

## CRITICAL CONSTRAINTS

### MUST
- Build RouteWeatherBadge component
- Render on each RouteAttachmentCard showing dominant condition
- 'Best for today' badge on highest-scored route with star icon and entrance animation (delay 400ms)
- Conditions score visible on each card
- 'Weather unavailable' indicator when data is missing
- Condition-to-visual mapping: clear=null (no badge), rain=danger colors, temp-high/low=warning/info, wind=muted

### NEVER
- Show a badge for clear/ideal conditions (clear = no badge)
- Hardcode weather data in the component
- Skip the entrance animation on 'Best for today'

### STRICTLY
- Animation delay for 'Best for today' badge: 400ms
- Badge colors must use semantic tokens matching the condition type

## SPECIFICATION

**Objective:** Add weather condition badges to route cards so riders can quickly compare weather impact across route options, with the best route highlighted.

**Success looks like:** Each route card shows its dominant weather condition as a badge. The highest-scored route displays a 'Best for today' badge with an animated entrance. Cards with missing weather data show an 'unavailable' indicator.

### Condition-to-Visual Mapping
| Condition | Icon | Color Token | Badge Text |
|-----------|------|-------------|------------|
| clear | none | none | (no badge) |
| rain-light | rain | semantic.color.warning | Light Rain |
| rain-moderate | rain | semantic.color.warning | Rain |
| rain-heavy | rain | semantic.color.danger | Heavy Rain |
| temp-hot | thermometer | semantic.color.warning | Hot |
| temp-cold | thermometer | semantic.color.info | Cold |
| wind-moderate | wind | semantic.color.muted | Windy |
| wind-high | wind | semantic.color.warning | High Wind |
| best-for-today | star | semantic.color.success | Best for today |

### Animation
- 'Best for today' badge uses Animated.FadeIn with 400ms delay
- Badge scales from 0.8 to 1.0 with spring easing
