## SPECIFICATION

**Objective:** Build a `ThinkingCard` component that shows a collapsible one-line chip in the chat transcript. While streaming, it animates through step summaries. When complete, tapping opens a `BottomSheetWrapper` with a full thinking timeline.

**Success looks like:**
- During agent planning: chip shows "Searching for Santa Cruz..." with pulsing dot, updates as new steps arrive
- After completion: chip shows "Thought for 5s" with chevron — tap opens bottom sheet
- Bottom sheet: vertical timeline of steps with icons (lightbulb/magnifying-glass/checkmark), summaries, details, and relative timestamps

## DESIGN

### Component Structure

```
ThinkingCard (CardProps)
├── CollapsedChip (Pressable)
│   ├── IconSymbol (lightbulb-on-outline)
│   ├── Text (summary or duration label, max 120 chars)
│   ├── PulsingDot (streaming only)
│   └── IconSymbol (chevron-down, complete + has steps only)
└── BottomSheetWrapper (visible when sheetOpen)
    ├── SheetHeader ("Agent Thinking" + duration + close)
    ├── StepsTimeline (ScrollView)
    │   └── StepRow[] (icon + summary + detail + timestamp)
    └── RawThinkingSection (collapsible, full content text)
```

### CollapsedChip Summary Logic

```typescript
function getChipSummary(
  status: CardProps['message']['status'],
  thinkingSteps: ThinkingStep[],
  startedAt: number,
  completedAt: number | null
): string {
  if (status === 'streaming' || status === 'running') {
    const lastStep = thinkingSteps[thinkingSteps.length - 1]
    return lastStep?.summary ?? 'Thinking...'
  }
  // Complete — show duration
  return formatDurationLabel(status, startedAt, completedAt)
}
```

### Duration Label

Reuse the `formatDurationLabel` pattern from `reasoning-card.tsx`. Capture `completedAt` locally on first observation of terminal status (same live-transition detection pattern).

### Accessibility
