## CRITICAL CONSTRAINTS

- Verify free tier blocks after 5 plans/month with helpful conversational message
- Verify pro tier is unlimited
- Verify usage display is accurate
- Test edge cases: month rollover, partial plans (aborted mid-generation)

### NEVER
- Bypass rate limiting for testing — test the actual production path
- Hard-code plan counts — use real Convex mutations

### STRICTLY
- Limit message must be conversational, not a raw error — e.g. "You've used all 5 free plans this month. Upgrade to Pro for unlimited planning."

## SPECIFICATION

**Objective:** Validate that the rate limiting system correctly enforces free tier limits, allows pro tier unlimited access, and displays accurate usage information with helpful messaging.

**Success looks like:** Plans 1-5 succeed on free tier, plan 6 shows a friendly limit message, month rollover resets the count, and pro tier has no limits.

```typescript
// convex/db/__tests__/planUsage.integration.test.ts
describe("Rate Limiting", () => {
  describe("Free Tier", () => {
    it("allows plans 1-5", async () => {
      for (let i = 1; i <= 5; i++) {
        const result = await createPlan(freeTierUser)
        expect(result.success).toBe(true)
      }
    })

    it("blocks plan 6 with friendly message", async () => {
      // ... create 5 plans
      const result = await createPlan(freeTierUser)
      expect(result.success).toBe(false)
      expect(result.message).toContain("5 free plans")
    })

    it("resets on month rollover", async () => {
      // ... create 5 plans in March
      // ... advance time to April
      const result = await createPlan(freeTierUser)
      expect(result.success).toBe(true)
    })

    it("does not count aborted plans", async () => {
      const plan = await startPlan(freeTierUser)
      await abortPlan(plan.id)
      const usage = await getUsage(freeTierUser)
      expect(usage.count).toBe(0)
    })
  })

  describe("Pro Tier", () => {
    it("allows unlimited plans", async () => {
      for (let i = 1; i <= 10; i++) {
        const result = await createPlan(proTierUser)
        expect(result.success).toBe(true)
      }
    })
  })
})
```
