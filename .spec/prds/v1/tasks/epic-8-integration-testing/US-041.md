## SPECIFICATION

**Objective:** Profile the app's critical rendering paths and fix any performance bottlenecks to ensure smooth UX under V1 load conditions.

**Success looks like:** Map renders 3 polylines without frame drops, chat scrolls smoothly at 60fps, session load completes in < 2s, and weather overlay toggle responds in < 500ms.
