# US-030: Implement savedRoutes Convex endpoints

> Task ID: US-030
> Type: FEATURE
> Priority: P0
> Estimate: 90 minutes
> Assignee: convex-implementer

## CRITICAL CONSTRAINTS

### MUST
- Create save mutation: from PlannedRouteOptionView with weather snapshot
- Create list query: by owner, paginated, newest first, excludes soft-deleted
- Create getById query: single route by ID
- Create rename mutation: update route name
- Create delete mutation: soft delete with deletedAt timestamp
- Create search query: name text match
- Include duplicate prevention (routePlanId + optionIndex)

### NEVER
- Hard delete routes — always soft delete with deletedAt
- Allow saving duplicates of the same route option
- Return soft-deleted routes in list or search queries

### STRICTLY
- Pagination uses Convex cursor-based pagination
- List ordered by _creationTime descending (newest first)
- Duplicate check: unique on (ownerId, routePlanId, optionIndex)

## SPECIFICATION

**Objective:** Build the Convex backend endpoints for the saved routes feature: save, list, get, rename, delete, and search.

**Success looks like:** Routes can be saved from the planning flow, listed with pagination, searched by name, renamed, and soft-deleted. Duplicates are prevented. Deleted routes are excluded from queries.

### Schema Shape
```typescript
savedRoutes: defineTable({
  ownerId: v.id("users"),
  routePlanId: v.id("routePlans"),
  optionIndex: v.number(),
  name: v.string(),
  description: v.optional(v.string()),
  polyline: v.string(), // encoded polyline
  conditionsScore: v.optional(v.number()),
  weatherSnapshot: v.optional(v.any()), // weather at save time
  origin: v.string(),
  destination: v.string(),
  distanceMeters: v.number(),
  durationSeconds: v.number(),
  deletedAt: v.optional(v.number()), // soft delete timestamp
})
  .index("by_owner", ["ownerId"])
  .index("by_owner_route", ["ownerId", "routePlanId", "optionIndex"])
```

### Endpoint Signatures
```typescript
// Mutations
save(args: { routePlanId, optionIndex, name, ... }) → savedRouteId
rename(args: { savedRouteId, name }) → void
remove(args: { savedRouteId }) → void  // soft delete

// Queries
list(args: { paginationOpts }) → PaginationResult<SavedRoute>
getById(args: { savedRouteId }) → SavedRoute | null
search(args: { query: string }) → SavedRoute[]
```
