## CRITICAL CONSTRAINTS

### MUST
- Star rating control with 1-5 tappable stars, 44px touch targets each
- Notes TextArea with add/edit/delete functionality
- Rating persists immediately on tap (optimistic update)
- Notes save via explicit Save button
- Rating displayed on list cards
- Note indicator icon displayed on list cards

### NEVER
- Allow note save without explicit button tap (no auto-save for notes)
- Use custom star icons — use standard star filled/outline from icon library

### STRICTLY
- All touch targets minimum 44px per Apple HIG

## SPECIFICATION

**Objective:** Build the rating and notes UI components in the route detail view, plus indicators on list cards.

**Success looks like:** User taps stars to rate a route with immediate feedback, can add/edit/delete notes via a TextArea, and sees both rating and note indicators on list cards.

### Interaction Notes
- Star rating: horizontal row of 5 star icons, filled up to current rating
- Tap any star to set rating (1-indexed)
- Rating fires mutation immediately (no Save button needed)
- Notes section: collapsed by default, "Add Note" button to expand
- Notes TextArea: multiline, max 500 characters, with Save and Cancel buttons
- Existing note: shows preview text with Edit and Delete actions
- List card: small star count (e.g. "4/5") and note icon indicator

### Code Pattern

```tsx
// StarRating component
export function StarRating({ value, onChange }: { value: number; onChange: (v: number) => void }) {
  return (
    <View style={{ flexDirection: 'row', gap: 4 }}>
      {[1, 2, 3, 4, 5].map((star) => (
        <Pressable
          key={star}
          onPress={() => onChange(star)}
          hitSlop={8}
          style={{ width: 44, height: 44, alignItems: 'center', justifyContent: 'center' }}
        >
          <StarIcon filled={star <= value} />
        </Pressable>
      ))}
    </View>
  )
}
```

### Anti-pattern (DO NOT)
Do NOT auto-save notes on blur or debounce — use explicit Save button. Do NOT use a modal for note editing — inline TextArea in the detail view.
