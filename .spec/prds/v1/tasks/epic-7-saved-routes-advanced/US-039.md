## CRITICAL CONSTRAINTS

### MUST
- Navigate button on route cards and route detail view
- Construct deep-link URLs from origin, destination, and waypoints
- Google Maps: `https://www.google.com/maps/dir/?api=1&origin=...&destination=...&waypoints=...`
- Waze: `https://waze.com/ul?ll=...&navigate=yes`
- Toast message before handoff ("Opening Google Maps..." / "Opening Waze...")
- Fallback if neither app is installed
- Use Linking.canOpenURL to check app availability

### NEVER
- Open a navigation app without showing the toast first
- Assume both apps are always installed — always check availability
- Pass more than 25 waypoints to Google Maps (API limit)

### STRICTLY
- Waypoints sampled evenly along route if > 25 points

## SPECIFICATION

**Objective:** Allow users to export a planned route to Google Maps or Waze for turn-by-turn navigation via deep links.

**Success looks like:** User taps Navigate on a route, sees available navigation apps, taps one, sees a brief toast, and the external app opens with the route pre-loaded. If neither app is installed, a fallback with coordinates is shown.

### Code Pattern

```typescript
// lib/navigation-export.ts
export function buildGoogleMapsUrl(
  origin: LatLng,
  destination: LatLng,
  waypoints: LatLng[]
): string {
  const sampled = sampleWaypoints(waypoints, 25)
  const waypointStr = sampled.map(w => `${w.lat},${w.lng}`).join('|')
  return `https://www.google.com/maps/dir/?api=1&origin=${origin.lat},${origin.lng}&destination=${destination.lat},${destination.lng}&waypoints=${waypointStr}`
}

export function buildWazeUrl(destination: LatLng): string {
  return `https://waze.com/ul?ll=${destination.lat},${destination.lng}&navigate=yes`
}

export function sampleWaypoints(waypoints: LatLng[], max: number): LatLng[] {
  if (waypoints.length <= max) return waypoints
  const step = waypoints.length / max
  return Array.from({ length: max }, (_, i) => waypoints[Math.round(i * step)])
}
```

### Anti-pattern (DO NOT)
Do NOT construct URLs inline in the component — extract to lib/navigation-export.ts for testability. Do NOT skip the Linking.canOpenURL check — iOS requires it for scheme detection. Do NOT pass raw polyline points as waypoints — sample them.
