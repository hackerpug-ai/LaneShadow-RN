## CRITICAL CONSTRAINTS

- Add rating (1-5 optional), notes (string optional), markedAsRidden (boolean optional), riddenAt (number optional) to savedRouteValidator
- Create rate, addNote, markAsRidden mutations
- Add rating filter to search query

### NEVER
- Make new fields required (additive optional only)
- Break existing saved route queries or mutations

## SPECIFICATION

**Objective:** Extend the saved routes schema with rating, notes, and ridden tracking fields, plus the mutations and query filters to support them.

**Success looks like:** Saved routes can be rated 1-5, annotated with notes, and marked as ridden. Rating filter returns only routes meeting the threshold.
