CODE PATTERN (Reference)

Existing RouteAttributes pattern:
```python
class RouteAttributes(BaseModel):
    """Extracted route attributes from LLM."""
    curvature_score: float = Field(ge=0.0, le=1.0, description="...")
    scenic_score: float = Field(ge=0.0, le=1.0, description="...")
    # ... 8 total score fields
    season: Optional[Season] = None
    road_surface: Optional[RoadSurface] = None
    primary_archetype_hint: Optional[Literal[...]] = None
```

New field addition:
```python
    # v2 addition
    extraction_confidence: Optional[float] = Field(
        default=None, ge=0.0, le=1.0,
        description="Model confidence score for this extraction (0.0-1.0)"
    )
```

Anti-pattern: Do NOT use `Field(default=None)` without `Optional[float]` annotation — Pydantic requires the Optional type hint.
