BACKGROUND
--------------------------------------------------------------------------------

**Problem:** The Route and EnrichedRoute dataclasses need 25+ new fields for designation, elevation, road quality, source provenance, and community scoring. A new RouteMention entity is needed for social source attribution.

**Why it matters:** Every subsequent epic depends on these model extensions — source tasks need source_refs, quality tasks need field_provenance, scoring tasks need new score fields, and community NLP needs RouteMention.

**Current state:** Route has 12 fields (route_id, name, state, source, centroid_lat, centroid_lng, length_miles, bounds_ne/sw). EnrichedRoute extends Route with 6 scores + classification + content fields. No RouteMention entity exists.

**Desired state:** Route has 18 new Optional fields (30 total). EnrichedRoute has 9 new Optional fields. RouteMention dataclass exists with 18 fields. All new fields default to None — backward compatible.

CODE PATTERN (Reference)
--------------------------------------------------------------------------------

Existing Route field pattern:
```python
@dataclass
class Route:
    route_id: str
    name: str
    state: str
    source: str
    centroid_lat: float
    centroid_lng: float
    length_miles: Optional[float] = None
    bounds_ne_lat: Optional[float] = None
    # ... more Optional fields with None default
```

New fields follow same pattern:
```python
    # NEW — extended fields (all Optional for backward compatibility)
    location: Optional[dict] = None              # GeoJSON Point
    description: Optional[str] = None
    rating: Optional[float] = None
    designation: Optional[str] = None
    source_url: Optional[str] = None
    source_refs: Optional[list] = None           # list[str]
    highway_number: Optional[str] = None
    elevation_gain_m: Optional[float] = None
    surface: Optional[str] = None                # paved|gravel|dirt|mixed
    aadt: Optional[int] = None
    aadt_median: Optional[float] = None
    aadt_max: Optional[float] = None
    pavement_iri: Optional[float] = None
    mention_frequency: Optional[float] = None
    source_priority: Optional[dict] = None       # dict[str, int]
    field_provenance: Optional[dict] = None       # dict[str, str]
    merged_at: Optional[str] = None              # ISO timestamp
    merge_count: Optional[int] = None
```

RouteMention pattern:
```python
@dataclass
class RouteMention:
    road_name: str
    sentiment_score: float           # -1.0 to 1.0
    confidence: float                # 0.0 to 1.0
    source: str
    source_authority: float
    post_url: str
    post_score: int
    processed_at: str                # ISO timestamp
    extraction_model: str
    extraction_cost: float
    highway_number: Optional[str] = None
    state: Optional[str] = None
    sentiment_label: Optional[str] = None   # positive|neutral|negative
    aspect_scores: Optional[dict] = None    # dict[str, float]
    attributes: Optional[dict] = None       # dict[str, bool]
    mention_date: Optional[str] = None
    route_id: Optional[str] = None          # FK after linking
```

Anti-pattern: Do NOT use `Optional[T]` without `= None` default — will break backward compatibility.
