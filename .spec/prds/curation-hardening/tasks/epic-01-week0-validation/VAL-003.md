BACKGROUND
--------------------------------------------------------------------------------

**Problem:** UC-SRC-03 depends on scraping `twtex.com`'s crowd-sourced Top 100 Motorcycle Roads list. Unlike government datasets, community sites often deploy WAF protection (Cloudflare, DataDome, Incapsula) or have restrictive Terms of Use that prohibit scraping. We don't know which category `twtex.com` falls into.

**Why it matters:** If `twtex.com` is WAF-protected or legally off-limits, we need to either abandon SRC-003 or pivot to a headless browser approach (Playwright) with much higher engineering cost. This validation is the Week 0 gate that determines whether SRC-003 is worth pursuing and in what form.

**Current state:** No legal or technical review has been conducted. The PRD assumes `twtex.com` is scrapable based on it being a small community site, but this has not been verified.

**Desired state:** A spike script at `scripts/curation/pipeline/sources/twtex_pilot/` that performs a live HTTP probe, reviews `robots.txt`, documents ToU findings, and produces a structured go/no-go decision report.

CODE PATTERN (Reference)
--------------------------------------------------------------------------------

Example implementation pattern:
```python
import requests
import json
import sys
from pathlib import Path

WAF_MARKERS = ["cloudflare", "incapsula", "datadome"]

def probe_waf(url: str, ua: str) -> dict:
    try:
        r = requests.get(url, headers={"User-Agent": ua}, timeout=30)
        body_lower = r.text.lower() if r.text else ""
        waf_detected = (
            r.status_code in (403, 429, 503)
            or any(m in body_lower for m in WAF_MARKERS)
        )
        return {
            "http_status_code": r.status_code,
            "waf_detected": waf_detected,
        }
    except requests.RequestException as e:
        return {"http_status_code": 0, "waf_detected": True, "error": str(e)}

def parse_robots(content: str, listing_path: str = "/top-100") -> dict:
    disallow_lines = [l.strip() for l in content.splitlines() if l.strip().lower().startswith("disallow:")]
    return {
        "robots_disallows_listing": any(listing_path in l for l in disallow_lines),
        "robots_disallow_lines": disallow_lines,
    }

def compute_go_no_go(report: dict) -> str:
    if report["waf_detected"] or report["robots_disallows_listing"] or not report["legal_ok"]:
        return "no-go"
    return "go"
```

Example test pattern:

def test_waf_detection(requests_mock):
    # GIVEN
