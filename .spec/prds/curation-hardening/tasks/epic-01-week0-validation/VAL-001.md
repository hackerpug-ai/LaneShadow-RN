BACKGROUND
--------------------------------------------------------------------------------

**Problem:** The curation-hardening PRD depends on a GLM NLP pipeline (UC-RIDER-03) that extracts route mentions and riding attributes from ~7.4M ADVRider forum posts and Reddit posts. The predicted cost is ~$370 for full backfill, dropping to ~$0.50/day incremental. Before committing to the full pipeline build (RID-003), we must prove that Claude 3 Haiku can actually hit the expected accuracy floor.

**Why it matters:** If Haiku accuracy falls below road name F1 0.75, the PRD calls for a fallback to sentence-transformer NLP (AD-002). The pilot is the Week 0 decision gate that controls whether RID-003 proceeds or branches to the fallback plan. Without this validation, the full pipeline build is speculative and could waste 2-3 weeks of effort.

**Current state:** `scripts/curation/pipeline/extraction/client.py` has a working Anthropic + Instructor client for structured extraction against route descriptions. `scripts/curation/pipeline/extraction/schema.py` defines `RouteAttributes`. No labeled dataset exists for forum post extraction, and no accuracy measurement has been run.

**Desired state:** A throwaway spike at `scripts/curation/pipeline/nlp/pilot/` with a 100-post labeled dataset, a `run_pilot.py` script executing Haiku extraction with retry logic, and a `results.json` producing PASS/FAIL status + exit code gating.

CODE PATTERN (Reference)
--------------------------------------------------------------------------------

Example implementation pattern (modelled on extraction/client.py):
```python
from anthropic import Anthropic
from instructor import from_anthropic
from pydantic import BaseModel
from tenacity import retry, stop_after_attempt, wait_exponential
import json
from pathlib import Path

class PilotExtraction(BaseModel):
    road_names: list[str]
    attributes: dict[str, str]  # scenery, twistiness, traffic, road_quality -> positive|negative|neutral|absent

client = from_anthropic(Anthropic())

@retry(stop=stop_after_attempt(3), wait=wait_exponential(min=1, max=10))
def extract_post(snippet: str) -> PilotExtraction:
    return client.messages.create(
        model="claude-3-haiku-20240307",
        max_tokens=1024,
        temperature=0,
        response_model=PilotExtraction,
        messages=[{"role": "user", "content": f"Extract road names and attribute sentiment: {snippet}"}],
    )
```
