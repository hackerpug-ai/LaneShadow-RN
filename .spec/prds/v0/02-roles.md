# User Roles

| Role | Description |
|------|-------------|
| Rider | Primary user - recreational motorcyclist who plans scenic rides in advance |
| System | Automated processes (route generation, weather fetching, data sync) |

## Rider Persona

**Primary**: Solo recreational motorcycle riders who plan rides ahead of time and prefer scenic, enjoyable routes over purely fastest paths.

**Characteristics**:
- Plans rides in advance (not real-time navigation)
- Values scenic quality over speed
- Wants weather/safety context before departing
- Returns to favorite roads and routes
- Rides 2-4 times per month (seasonal)

**Jobs to Be Done**:
1. When planning a weekend ride, I want to see weather conditions along the route so I can decide if it's safe
2. When I discover a great road, I want to save it so future routes can include it
3. When I return from a ride, I want to rate it so I remember which routes were best
4. When planning the next ride, I want to browse my history so I can revisit favorites
