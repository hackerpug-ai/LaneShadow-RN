§01 · Purpose

Four meanings, one shape.

The pill shape is neutral. Meaning is injected through semantic wrapper molecules: LSTagPill informs without inviting interaction; LSFilterChip toggles a query parameter; LSSuggestionChip primes a prompt with a single tap; LSWeatherBadge communicates condition through tinted color and a leading icon. All four compose UC-ATM-06 LSPill — the shape primitive — and never recreate the pill radius inline.

Heights follow the three-tier dimension token: sm 24pt, md 32pt, lg 40pt. SuggestionChip is fixed at md (32pt) per spec. WeatherBadge ships sm and md.

§06 · Rationale

Design decisions.

Semantic not stylistic

The four pill variants share the same border-radius token but carry different interaction semantics. LSTagPill has no press handler; LSFilterChip carries toggle state; LSSuggestionChip fires once then primes; LSWeatherBadge is decorative. Separating these prevents misuse — a filter chip cannot be accidentally treated as decorative.

  • border-radius 999px — token radius.pill from UC-TOK-03
  • heights 24 / 32 / 40pt — sm/md/lg sizing tokens
  • glass backing on TagPill + SuggChip — float above map layer

AC Traceability — 06-uc-mol.md §UC-MOL-05

  • LSTagPill renders glass surface + pin icon + label on both platforms
  • LSFilterChip selected shows signal.default background; unselected shows surface.card
  • LSSuggestionChip tap fires onTap exactly once; primed state shown post-tap
  • LSWeatherBadge all 6 conditions × 2 sizes render correct tint + fg + border + icon
  • All molecules compose from LSPill + LSIcon + LSText; raw shape implementations fail inspection
  • Stories: Pill Semantics / Tag, Filter, Suggestion, Weather — all variants covered
