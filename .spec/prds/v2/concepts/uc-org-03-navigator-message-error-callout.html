§01 · Purpose

Voice of the Navigator.

LSNavigatorMessage is the branded AI callout used whenever the Navigator produces a response with optional route attachments. A copper-tinted compass chip identifies the speaker; the body is set in Newsreader italic — opinion typography — to distinguish the Navigator's voice from the rider's UI. Attachments stack below as compact LSRouteAttachmentCards, first card selected.

LSInlineErrorCallout is the recovery callout for the ErrorScreen. It swaps the copper accent for a warning amber, adds an optional detail line, and shows LSSuggestionChip rows so the rider can recover without retyping a prompt.

§06 · Rationale

Opinion type, glass surface.

Design Decision

The body uses Newsreader (opinion typography) to distinguish the Navigator's voice from UI chrome. Every other UI element uses Geist; when the Navigator speaks, the typeface shifts — making it unmistakably a voice, not a label.

The compass chip inside a tinted pill signals authorship without using an avatar or logo. It's minimal, brand-specific, and immediately scannable.

Error callouts use an amber warning accent instead of copper signal so riders recognize them as recoverable error states — different from recommendations.
