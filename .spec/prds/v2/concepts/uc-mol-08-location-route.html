§01 · Purpose

Context before the prompt.

LSLocationContextBar slots above the suggestion row in LSChatInput, surfacing two pieces of ambient context — your location and whether it is derived automatically or set manually. Tapping the mode pill fires onModeChange, allowing the user to override GPS.

LSRouteAttachmentCard is the Navigator's route result surface inside chat messages and catalog views. The 3px leading stripe encodes the route variant (best / alt1 / alt2) using the color.route.* polyline palette. The best badge, scenic meter, weather badge, and favorite flag compose from UC-ATM-07 and UC-ATM-10 atoms.

§06 · Rationale

Design decisions.

3px stripe encodes variant identity

The leading stripe uses the same polyline palette as the map layer — route.best copper, route.alt1 sage-green, route.alt2 slate. This creates a persistent visual thread between the card in the chat stream and the route drawn on the map, so the rider always knows which card corresponds to which polyline without reading the title.

  • border-radius 10pt — matches UC-ATM-05 radius.md token
  • 3px stripe — thinner than a border, decorative not structural
  • selected bg — color-mix(surface.card, signal.default, 8%) keeps legibility
  • compact=true — no badges, tighter padding 8/10px for list contexts
