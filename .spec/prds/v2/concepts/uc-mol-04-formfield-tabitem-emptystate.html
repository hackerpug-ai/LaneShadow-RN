§01

Purpose & Atomic Composition

LSFormField wraps LSTextField (UC-ATM-03) with a label and helper/error text row, giving screens a single API for labeled inputs without re-implementing the label-input-error stack layout. LSTabItem composes LSIcon (UC-ATM-10) with a label and a 2px selected-indicator bar — one tab cell in a bottom navigation bar. LSEmptyState is a centered composition using an illustration slot (sizing.icon.xl = 32pt), title in typography.ui.title.md, body in typography.ui.body.md, and an LSButton (UC-ATM-02) primary action.

All three molecules must compose through atoms exclusively. No raw Text with a literal color. No raw Box with a literal corner radius. Every value must trace back to a token.

§05

Motion Recipes

tabSwitch — when the selected LSTabItem changes, the indicator bar slides horizontally. Duration 200ms, easing: ease-out. Icon + label color cross-fade simultaneously. No bounce — tabs are utility navigation, not expressive moments.

formFieldError — when an error state appears, the error text animates in with opacity 0→1 + translateY(-2px)→0 in 160ms ease-out. The input border color transition is 120ms ease-in. No shake — shake patterns are disorienting on mobile.

emptyStateEnter — when an empty state is revealed (e.g., after a filter removes all results), the icon and text stack fade in sequentially with 40ms stagger. Overall duration: 280ms. Icon first, then title, then body, then button.

§06

Rationale + AC Traceability

LSFormField uses spacing.2 (4px) between label, input, and helper rather than the more common 8px gap because the visual grouping of label-to-input is intentionally tight — the label should read as a part of the input, not a separate element. Screens using multiple form fields use spacing.5 (16px) between fields.

The 2px LSTabItem indicator bar at the top rather than bottom follows iOS convention (top indicators) rather than Material Design (bottom) — LaneShadow's primary platform is iOS. The bar is fixed at 24pt wide rather than full-cell to avoid harsh hard-edge movement during tab transitions.

LSEmptyState uses a framed 64pt slot for icons rather than bare floating icons because floating icons at 32pt look weightless and easy to miss. The frame adds presence without introducing a full illustration asset requirement.
