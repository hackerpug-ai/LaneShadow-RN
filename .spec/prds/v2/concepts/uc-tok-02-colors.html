§01

Purpose

The Copper palette is a warm-paper topographic aesthetic rooted in the physical world of road maps and navigation instruments. Warm-paper neutrals define the day surfaces; warm-ink neutrals define the night. The copper signal color at #EE7C2B is the single accent hue — it carries the brand, the "best route" indicator, the interactive focus state, and the Navigator identity.

All colors are declared in semantic namespaces so that a designer changing the weather palette for accessibility does not touch surface or signal tokens. No downstream component ever references a raw hex value.

§03

Light vs Dark

Every semantic token has both light and dark values. The dark mode invert is not a simple luminance flip — the warm-ink palette preserves the earthy, topographic character of the Copper system. The copper signal color remains identical in both modes; only surface and content tokens swap.

§05

Motion

Color tokens carry no motion behavior. The scrim color token (surface.scrim) is consumed by sheet dismiss animations, but the easing and duration are declared in UC-TOK-04 motion recipes. Color is static data; motion is a separate concern.

§06

Rationale & references

Design decisions

The warm-paper palette was chosen over a cool-gray neutral to evoke physical road maps and printed cartography. The warmth reinforces the Copper signal color's analogous relationship and creates a cohesive system that reads "navigator instrument" rather than "generic app."

Weather colors are intentionally distinct from route colors and from the copper signal, so that a map surface showing all three simultaneously reads clearly without visual collision.

PRD acceptance criteria

  • AC-01 Ten semantic namespaces in semantic.tokens.json
  • AC-02 All colors from designs.html present in token file
  • AC-03 schema validation passes with zero errors
  • AC-04 Each token has $type: "color" and valid $value
  • AC-05 Weather tints produce distinct pill backgrounds in both themes
  • AC-06 role.agent.* resolves to copper signal (brand continuity)
  • AC-07 pnpm tokens:validate exits zero errors
