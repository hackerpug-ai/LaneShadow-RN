# User Roles

| Role | Description |
|------|-------------|
| **Domain Expert** | Curates quality pools, defines taste profiles, sets quality criteria, validates recommendations. Has deep domain knowledge and cultural context. |
| **System Administrator** | Configures quality thresholds, manages taste profiles, monitors system performance, oversees expert curation workflow. |
| **End User** | Receives recommendations, provides explicit feedback (helpful/not helpful), benefits from improved curation quality. |
| **Developer** | Maintains the curation system, integrates with LLM agents, extends metadata schemas, builds custom taste profiles. |
| **Analytics Viewer** | Reviews recommendation quality metrics, pool performance data, and expert efficiency dashboards. |
