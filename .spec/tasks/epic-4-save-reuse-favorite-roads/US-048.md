# US-048: Show favorite inclusion indicator on generated routes

> Task ID: US-048
> Type: FEATURE
> Priority: P2
> Estimate: 90 minutes
> Assignee: ui-developer

## CRITICAL CONSTRAINTS

### MUST
- Read components/planning/route-option-card.tsx for existing patterns
- Display visual indicator when favorite segment is in route
- Show which favorites were included
- Follow semantic theme for indicator colors

### NEVER
- Break existing route card layout
- Show indicators when no favorites were used
- Create complex UI (simple badge/pill only)

### STRICTLY
- Indicator shows on route cards that include favorites
- Show count: "X favorite roads included"
- Use semantic color (success/primary) for indicator

## SPECIFICATION

**Objective:** Display visual indicators on route cards showing which favorite roads influenced each route option

**Success looks like:** Users can see which routes include their favorite roads via clear visual indicators

### Interaction Notes
- Badge appears in same row as weather badges
- Badge color: semanticColors.primary.default
- Badge text: "X favorites" or "1 favorite"
- Tap to expand: show list of favorite names included
- Expandable inline (no new screen)

### Code Pattern

Source: components/planning/route-option-card.tsx

```typescript
import { Badge } from '../../components/ui/badge'
import { Heart } from '../../components/ui/icon-symbol'

type RouteOptionCardProps = {
  // ... existing props
  favoriteCount?: number
  favoriteNames?: string[]
}

export const RouteOptionCard: React.FC<RouteOptionCardProps> = ({
  favoriteCount = 0,
  favoriteNames = [],
  ...props
}) => {
  const [showFavorites, setShowFavorites] = useState(false)

  return (
    <Card>
      {/* Existing content */}

      {/* Badges row */}
      <View style={styles.badgesRow}>
        {/* Existing wind/rain/temp badges */}

        {favoriteCount > 0 && (
          <TouchableOpacity onPress={() => setShowFavorites(!showFavorites)}>
            <Badge variant="primary" size="small">
              <Heart size={12} />
              <ThemedText variant="caption">
                {favoriteCount} favorite{favoriteCount > 1 ? 's' : ''}
              </ThemedText>
            </Badge>
          </TouchableOpacity>
        )}
      </View>

      {/* Expandable favorite list */}
      {showFavorites && favoriteNames.length > 0 && (
        <View style={styles.favoriteList}>
          {favoriteNames.map(name => (
            <ThemedText key={name} variant="caption">
              • {name}
            </ThemedText>
          ))}
        </View>
      )}
    </Card>
  )
}
```

### Anti-pattern (DO NOT)
- Don't show indicator when favoriteCount=0
- Don't create complex list view (inline expansion only)
- Don't break existing badge row layout
