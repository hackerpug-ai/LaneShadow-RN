> Task ID: US-051
> Type: FEATURE

> Assignee: convex-implementer

## CRITICAL CONSTRAINTS

### MUST
- Use Google Routes API for deviation cost calculation
- Implement nearest-point-on-polyline algorithm
- Classify on-route if within 500m threshold
- Handle API failures gracefully

### NEVER
- Block route generation on waypoint calculation failures
- Cache deviation results without route geometry fingerprint
- Modify route geometry during calculation

### STRICTLY
- Deviation calculation must complete in <5 seconds
- Return structured detourInfo object or on-route classification
- All distance/time values in meters/seconds

## SPECIFICATION

**Objective:** Calculate whether a waypoint is on-route or off-route, and compute deviation costs for off-route waypoints.

**Success looks like:** Service accepts waypoint + route geometry, returns classification and detour costs if applicable.
