## CRITICAL CONSTRAINTS

### MUST
- Display weather conditions for each route segment/leg
- Show distance/duration per segment alongside weather
- Support expand/collapse interaction
- Highlight concerning segments with warning styling

### NEVER
- Fetch additional data on expand - use existing overlay data
- Show more than 3 segments initially (collapse remainder)

### STRICTLY
- Segment order must match route direction
- Use Collapsible component from UI library

## SPECIFICATION

**Objective:** Create an expandable detail view showing segment-by-segment weather breakdown for the selected route.

**Success looks like:** Users can tap to expand and see weather conditions for each leg of their route, helping them identify specific problem areas.
