## CRITICAL CONSTRAINTS

### MUST
- Create `DateRangePicker` component at `components/ui/date-range-picker.tsx`
- Support preset options: "Last week", "Last month", "Last 3 months"
- Support a "Clear" / "All time" option that removes the date filter
- Use semantic theme tokens for ALL visual properties
- Emit `{ afterDate?: number; beforeDate?: number }` via `onDateRangeChange` callback

### NEVER
- Import external date picker libraries (presets only for MVP)
- Use hardcoded colors, spacing, or font sizes
- Include custom date range input (presets only in this task)
- Make network calls from within the component

### STRICTLY
- Presets calculate dates relative to `Date.now()` at selection time
- "Last week" = 7 days ago to now; "Last month" = 30 days ago to now; "Last 3 months" = 90 days ago to now
- Selected preset must be visually highlighted (active state)
- "All time" or clear selection emits `{ afterDate: undefined, beforeDate: undefined }`

## SPECIFICATION

**Objective:** Create a chip-style date range picker with preset options for filtering saved routes by creation date.

**Success looks like:** A horizontal row of selectable chips ("All time", "Last week", "Last month", "Last 3 months"). Tapping a chip highlights it and emits the corresponding date range. Tapping "All time" clears the filter.

### Interaction Notes
- Layout: horizontal ScrollView of chip-style buttons
- Active chip: `semantic.color.primary.default` background, `semantic.color.onPrimary.default` text
- Inactive chip: `semantic.color.surfaceVariant.default` background, `semantic.color.onSurface.default` text
- Chip shape: `semantic.radius.full` (pill shape)
- Chip padding: `semantic.space.sm` vertical, `semantic.space.md` horizontal
- Gap between chips: `semantic.space.sm`
- The row should be horizontally scrollable if chips overflow

### Code Pattern
Source: Chip-based filter pattern

```typescript
type DateRangePreset = 'all' | 'week' | 'month' | '3months'

type DateRange = {
  afterDate?: number
  beforeDate?: number
}

type DateRangePickerProps = {
  onDateRangeChange: (range: DateRange) => void
  testID?: string
}

const PRESETS: Array<{ key: DateRangePreset; label: string; daysBack?: number }> = [
  { key: 'all', label: 'All time' },
  { key: 'week', label: 'Last week', daysBack: 7 },
  { key: 'month', label: 'Last month', daysBack: 30 },
  { key: '3months', label: 'Last 3 months', daysBack: 90 },
]
```
