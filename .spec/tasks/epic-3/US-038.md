### Code Pattern
Source: components/ui/button.tsx

```typescript
// BEFORE (anti-pattern):
<View onTouchEnd={handlePress}>

// AFTER (correct):
<Pressable
  onPress={handlePress}
  accessibilityRole="button"
  accessibilityLabel={label}
  android_ripple={{ color: semantic.color.primary.container }}
>
```

### Anti-pattern (DO NOT)
Do NOT use `onTouchEnd`, `onTouchStart`, or other low-level touch handlers for button interactions. Do NOT use `TouchableOpacity` -- use `Pressable`.

  - `accessibilityRole="button"` on all clickable items
