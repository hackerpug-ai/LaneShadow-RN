## DESIGN

### Installation Command

```bash
npm install @rnmapbox/maps@^10.1.0
```

### Package.json Entry

```json
{
  "dependencies": {
    "@rnmapbox/maps": "^10.1.0"
  }
}
```

### Peer Dependencies

@rnmapbox/maps requires:
- `react`: >=16.6.0
- `react-native`: >=0.60.0

### Platform-Specific Setup

**iOS:**
- Requires `npx pod-install` after npm install
- Adds Mapbox.framework to Xcode project

**Android:**
- Requires Gradle sync
- Adds Mapbox SDK to Android dependencies

## DESIGN NOTES

### Why @rnmapbox/maps?

- Official React Native wrapper for Mapbox SDK
- Supports both iOS and Android
- Compatible with Expo (bare workflow)
- Active maintenance and community support

### Version Pinning

Using `^10.1.0` allows:
- Patch updates automatically (10.1.1, 10.1.2, etc.)
- Minor updates within 10.x series
- Prevents breaking changes from 11.x

### Installation Verification

Both platforms must build successfully:
- **iOS:** CocoaPods installs native frameworks
- **Android:** Gradle downloads native SDK

Failure on either platform indicates installation issue.
