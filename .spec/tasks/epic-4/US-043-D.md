BACKGROUND
--------------------------------------------------------------------------------

**Problem:** No existing action sheet for naming and saving favorite road segments.

**Why it matters:** Action sheet is the confirmation UI for saving favorites. Users need clear input validation and feedback.

**Current state:** BottomActionSheet component exists but no layout spec for save favorite form.

**Desired state:** Complete action sheet specification with header, input, validation states, and button states.

--------------------------------------------------------------------------------
DESIGN REQUIREMENTS
--------------------------------------------------------------------------------

### Layout
```
┌────────────────────────────────────────┐
│ [Heart Icon] Save as Favorite          │  <- Header
├────────────────────────────────────────┤
│ Name this road                         │  <- Input Label
│ ┌────────────────────────────────────┐ │
│ │ e.g., Pacific Coast Highway     [X]│ │  <- Input
│ └────────────────────────────────────┘ │
│ 25/100                                 │  <- Character Counter
├────────────────────────────────────────┤
│ [Cancel]              [Save]           │  <- Buttons
└────────────────────────────────────────┘
```

### Component Specs

**BottomActionSheet**:
- Snap point: 40% height
- Stack behavior: push (can stack on other sheets)
- Backdrop: opacity 0.5, pressBehavior: "close"
- Handle: hidden (no drag handle needed)

**Header**:
- Icon: Heart (MaterialCommunityIcons "heart")
- Icon size: 24px
- Icon color: semantic.color.primary.default
- Title: "Save as Favorite"
- Title variant: titleMedium
- Title color: semantic.color.onSurface.default
- Padding: 16px horizontal, 16px top
- Gap: 8px (between icon and title)

**Input Field**:
- Component: TextInput (outlined mode)
- Label: "Name this road"
- Placeholder: "e.g., Pacific Coast Highway"
- Max length: 100 characters
- Auto-focus: true
- Auto-capitalize: sentences
- Border color:
  - Default: semantic.color.border.default
  - Focus: semantic.color.primary.default
  - Error: semantic.color.danger.default
- Padding: 12px
- Border radius: semantic.radius.md (8px)

**Character Counter**:
- Text: "X/100" (e.g., "25/100")
- Color:
  - Default: semantic.color.onSurface.subtle
  - Near limit (>80): semantic.color.warning.default
  - At limit (100): semantic.color.danger.default
- Variant: bodySmall
- Alignment: right
- Margin top: 4px

**Buttons**:
- Layout: Horizontal row, 50/50 split
- Gap: 12px
- Height: 48px each
- Border radius: semantic.radius.md (8px)

**Cancel Button**:
- Variant: outline
- Text: "Cancel"
- Color: semantic.color.onSurface.default
- State: Always enabled

**Save Button**:
- Variant: default (filled)
- Text: "Save"
- Color: semantic.color.onPrimary.default
- Background: semantic.color.primary.default
- States:
  - Disabled (empty name): opacity 0.5, not interactive
  - Enabled (valid name): full opacity, interactive
  - Loading: shows spinner, text "Saving..."

### Validation States

**State 1: Empty (Initial)**
- Input: empty
- Counter: "0/100"
- Save button: disabled
- No error message

**State 2: Typing (1-99 chars)**
- Input: has text
- Counter: "X/100" (dynamic)
- Save button: enabled
- No error message

**State 3: At Limit (100 chars)**
- Input: 100 characters
- Counter: "100/100" (danger color)
- Save button: enabled
- No error message (truncation prevents overflow)

**State 4: Duplicate Name**
- Input: matches existing favorite name
- Save button: disabled
- Error message: "You already have a favorite with this name"
- Error color: semantic.color.danger.default

**State 5: Saving (Loading)**
- Input: disabled
- Buttons: disabled
- Save button: shows spinner
- Text: "Saving..."

**State 6: Success**
- Action sheet dismisses
- Success toast: "Saved 'Pacific Coast Highway' to favorites"
- Parent component refreshes
