--------------------------------------------------------------------------------
BACKGROUND
--------------------------------------------------------------------------------

**Problem:** No existing card component with thumbnail + text + delete button layout.

**Why it matters:** Cards are the primary UI element for displaying favorites in the settings list. Users need to identify favorites by name and visual preview.

**Current state:** Card component exists but no layout spec for favorite road card with thumbnail.

**Desired state:** Complete card specification with layout, states, empty state, and accessibility.

--------------------------------------------------------------------------------
DESIGN REQUIREMENTS
--------------------------------------------------------------------------------

### Layout
```
┌──────────────────────────────────────────────┐
│ [Thumbnail]  Name              [Delete Icon] │
│  80×80       Location                      │
└──────────────────────────────────────────────┘

Dimensions:
- Card height: 96px
- Card width: 100% (full width of container)
- Padding: 16px
- Border radius: 12px
- Gap between elements: 12px

Element Sizing:
- Thumbnail: 80×80px (fixed)
- Text container: flex 1 (takes remaining space)
- Delete button: 44×44px touch target
```

### Component Specs

**Card Container**:
- Component: Pressable (React Native)
- Background: semantic.color.surface.default
- Border: 1px, semantic.color.border.default
- Border radius: semantic.radius.lg (12px)
- Padding: 16px
- Layout: row (horizontal)
- Gap: 12px

**Thumbnail (RouteThumbnail)**:
- Component: RouteThumbnail (existing)
- Width: 80px
- Height: 80px
- Border radius: semantic.radius.md (8px)
- Props: bounds={favoriteRoad.bounds}
- Accessibility label: "Road map preview"

**Text Container**:
- Layout: column (vertical)
- Gap: 4px
- Flex: 1 (takes remaining space)
- Max width: calc(100% - 80px - 44px - 48px)

**Name Text**:
- Variant: titleMedium (16px, 600 weight)
- Color: semantic.color.onSurface.default
- Max lines: 2
- Ellipsize: tail
- Accessibility: Part of card label

**Location Text** (Subtitle):
- Variant: bodySmall (14px, 400 weight)
- Color: semantic.color.onSurface.subtle
- Max lines: 1
- Ellipsize: tail
- Format: "{nearest city, state}" or "Unknown location"
- Accessibility: Part of card label

**Delete Button**:
- Component: Pressable with IconSymbol
- Icon: "delete" (MaterialCommunityIcons)
- Icon size: 20px
- Icon color: semantic.color.error.default
- Touch target: 44×44px
- Background: transparent
- Border radius: semantic.radius.sm (4px)
- Padding: 12px (to achieve 44×44 touch target)
- Accessibility label: "Remove {name} from favorites"

### States

**State 1: Default (Idle)**
- Background: semantic.color.surface.default
- Border: 1px solid semantic.color.border.default
- Opacity: 1.0
- Shadow: none

**State 2: Pressed (Card Tapped)**
- Background: semantic.color.surfaceVariant.default
- Border: 2px solid semantic.color.primary.default
- Opacity: 0.8
- Shadow: semantic.elevation.sm

**State 3: Delete Pressed**
- Delete button background: rgba(error, 0.1)
- Delete icon color: semantic.color.error.dark
- Button pressed feedback

**State 4: Disabled (Loading)**
- Opacity: 0.5
- Not interactive
- Shows spinner if delete in progress

### Accessibility

**Card Label**:
- Format: "Favorite road {name}, {location}"
- Example: "Favorite road Alpine Loop, Colorado"
- Role: button
- Action: Opens favorite details (future) or does nothing (current)

**Delete Button Label**:
- Format: "Remove {name} from favorites"
- Example: "Remove Alpine Loop from favorites"
- Role: button
- Action: Triggers delete confirmation

**Focus Order**:
1. Card (if pressable for details)
2. Delete button

**Screen Reader Announcements**:
- Card: "{name}, favorite road, {location}, double tap to view details"
- Delete: "Remove {name} from favorites, double tap to delete"

### Empty State (for Settings List)

**Layout**:
```
        [Heart Icon + Road Icon]
    No favorite roads yet
  Save a road segment to see it here
        [Plan a ride] [CTA Button]
```

**Component Specs**:
- Icon: MaterialCommunityIcons "heart" + "road-variant" (stacked)
- Icon size: 64px
- Icon color: semantic.color.onSurface.muted
- Title: "No favorite roads yet"
- Title variant: titleMedium
- Subtitle: "Save a road segment to see it here"
- Subtitle variant: bodyMedium
- CTA button: "Plan a ride"
- Button variant: default
- Layout: center aligned, vertical column
- Gap: semantic.space.lg (16px)
